# DAET_Lagrange-EQ
Lagrange EQ is a high-quality parametric EQ plug-in by DAET, featuring large-window bidirectional IIR processing for a linear-phase-like EQ experience. It includes 8 bands, per-band Stereo/Mid/Side/Left/Right targeting, x2 oversampling, spectrum analyzer, A/B, Undo/Redo, and user presets.
